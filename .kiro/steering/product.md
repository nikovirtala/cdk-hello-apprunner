# Product Overview

This is a minimal AWS CDK example project that demonstrates deploying an AWS App Runner service.

The project deploys the `hello-app-runner` container image from Amazon ECR Public Gallery, showcasing a simple containerized web application running on AWS App Runner - a fully managed service for deploying containerized web applications and APIs.

## Key Features

- Minimal CDK infrastructure code
- Uses AWS App Runner for serverless container deployment
- Fetches public container image from ECR Public Gallery
- Outputs the service URL for easy access
